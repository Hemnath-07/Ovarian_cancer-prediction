## 📌 Project Overview  

Ovarian Cancer is one of the most serious diseases affecting women, often diagnosed at later stages. Early prediction can help in improving survival rates.

This project is a **Machine Learning-based web application** that predicts the likelihood of ovarian cancer using input medical parameters. The application is built using **Streamlit**, making it simple and interactive for users.

---

## 🎯 Objectives  

- Develop a model to predict ovarian cancer  
- Provide an easy-to-use web interface  
- Support early detection using AI  
- Demonstrate real-world ML application in healthcare
